# What is Programming?
==> Programming is the process of giving precise instructions to a computer to solve a problem.

Programming ≠ Language
Many people think:
Programming = Python
Programming = Java
Programming = PHP
No.
Programming is:
Problem
   ↓
Logic
   ↓
Language
   ↓
Computer


# Why Python?
==> Because almost every AI ecosystem is built around Python.

Examples:
PyTorch
TensorFlow
Transformers
LangChain
LlamaIndex
FastAPI
Hugging Face

All support Python first.
